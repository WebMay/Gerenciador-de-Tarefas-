# Gerenciador de Tarefas

Aplicação em Python desenvolvida para gerenciar tarefas via terminal.

## Funcionalidades
- Adicionar tarefas
- Visualizar tarefas
- Atualizar tarefas
- Marcar como concluída
- Remover tarefas concluídas

## Tecnologias
- Python 

## Objetivo
Projeto desenvolvido para praticar lógica de programação, estruturas de dados e controle de fluxo.
